## Diretivas obrigatórias (este comando)

1. **Leia e aplique por completo** a skill **`cria-criterios`**: `~/.cursor/skills/cria-criterios/SKILL.md`.
2. **Input:** link do Figma (com `node-id` quando possível) e lista das funcionalidades/telas a cobrir.
3. **MCP Figma** (`plugin-figma-figma`): `get_design_context` e `get_screenshot` — ver schema das tools no projeto antes de chamar.
4. **Entrega:** por funcionalidade — título, **Objetivo**, **Critérios de aceite** (até 5, testáveis); português; detalhe de colunas, status e ordem visual quando existirem no design.

Se o utilizador pedir **requison**, delegar conforme a skill indica; caso contrário, produzir no mesmo formato no chat.
